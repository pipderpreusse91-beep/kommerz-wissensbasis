# die

**Englischer Begriff:** They / The System
**Deutsche Entsprechung:** die Anderen / das System

## Interne Definition
Unbenannte Akteure (Staat, System, Behörden), die Handlungen vornehmen, die normalerweise unter der Haftung der Person stattfinden
