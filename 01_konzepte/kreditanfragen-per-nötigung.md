# Kreditanfragen per Nötigung

**Englischer Begriff:** Extortion / Coercion (als Charakterisierung behördlicher Forderungen)
**Deutsche Entsprechung:** Erpressung / Nötigung

## Interne Definition
Behördliche Geldforderungen, die intern als unfreiwillige Kreditvergabe durch den Betroffenen umgedeutet werden, die durch Zwang durchgesetzt werden
