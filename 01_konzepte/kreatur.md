# Kreatur

**Englischer Begriff:** Creature / Entity / Strawman (kontextabhängig)
**Deutsche Entsprechung:** Geschöpf / Kunstperson / abhängige Entität

## Interne Definition
Eine durch höhere Instanz geschaffene Entität (juristische Person, Behörde, Agent), die vom Schöpfer abhängig ist und nur eingeschränkte Eigenständigkeit besitzt
