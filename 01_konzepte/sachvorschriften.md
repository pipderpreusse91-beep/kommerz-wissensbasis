# Sachvorschriften

**Englischer Begriff:** substantive rules / domestic law
**Deutsche Entsprechung:** Materiellrechtliche Vorschriften / Direktnormen

## Interne Definition
Materiellrechtliche Normen eines Staates, die direkt auf den Sachverhalt angewandt werden, ohne weiteren Verweis auf anderes Recht (im Gegensatz zu Kollisionsnormen).
