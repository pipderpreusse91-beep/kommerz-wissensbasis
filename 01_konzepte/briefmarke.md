# Briefmarke

**Englischer Begriff:** Postage stamp (as commercial instrument)
**Deutsche Entsprechung:** Freimachung als kommerzielles Angebot-Kennzeichen

## Interne Definition
Kennzeichen der kommerziellen Natur einer Zustellung; rechtliches Signal, dass ein Angebot vorliegt, das angenommen oder verweigert werden kann
