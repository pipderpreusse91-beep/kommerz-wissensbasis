# Kommerz

**Englischer Begriff:** Commerce
**Deutsche Entsprechung:** Kommerzielles System / Handelssystem

## Interne Definition
Nicht im alltäglichen Sinne von Handel, sondern als ein System oder eine Ebene, deren Verständnis mentale Voraussetzung für tiefere Lösungen ist – wahrscheinlich im Sinne des kommerziellen Rechtssystems oder UCC-basierter Konzepte
