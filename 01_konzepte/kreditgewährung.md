# Kreditgewährung

**Englischer Begriff:** Credit issuance / Acceptance for Value
**Deutsche Entsprechung:** Kreditausstellung / Wertannahme

## Interne Definition
Der durch Unterschrift vollzogene Akt, durch den ein Mensch dem System Wert/Kredit zur Verfügung stellt und damit Geld in Umlauf bringt
