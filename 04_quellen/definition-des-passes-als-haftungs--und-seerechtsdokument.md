# Definition des Passes als Haftungs- und Seerechtsdokument

## Kernaussage
Der Text definiert einen Pass primär als Versicherungsdokument zur Haftungstransferierung zwischen Staaten und als Seerechtsdokument.

## Axiome
- [[Ein Pass ist ein Versicherungsdokument.]]
- [[Ein Pass ist ein Passierschein im Seerecht.]]
- [[Die Funktion eines Passes ist auf diese Aspekte beschränkt ("mehr ist das nicht").]]

## Begriffe

### PASS
**Englisch:** PASS
**Deutsch:** Reisedokument / Versicherungsdokument
Versicherungsdokument, das dem Besucherland die Gelegenheit gibt, bei Schäden im Ausland die Haftung auf das Ausstellerland der Person geltend zu machen

### Seerecht
**Englisch:** Maritime Law
**Deutsch:** Seerecht
Rechtsgebiet, in dem der Pass als Passierschein fungiert

### Pass port
**Englisch:** pass port
**Deutsch:** Passierschein
Etymologische Referenz zur Begründung des Passierschein-Status im Seerecht


## Argumentationsketten
- Axiom 1 (Pass = Versicherungsdokument) → Schluss: Besucherland kann Haftung auf Ausstellerland übertragen (weil Pass diese Gelegenheit gibt)
- Axiom 2 (Pass = Passierschein Seerecht) → Schluss: Etymologie "pass port" bestätigt diese Definition (weil pass port als Referenz genannt wird)

## Offene Stellen
- Keine logische Brücke zwischen der Definition als "Versicherungsdokument" und der Definition als "Seerechtsdokument" wird geschlagen.
- Die Aussage "mehr ist das nicht" schließt andere Funktionen aus, ohne zu begründen, warum diese Definition exklusiv ist.
- Der Begriff "Haftung auf das Ausstellerland der PERSON" ist rechtlich unpräzise definiert (staatliche Haftung vs. persönliche Haftung).
