# Der NAME als Falle: Systemteilnahme als Zwang

## Kernaussage
Der staatlich registrierte NAME gehört einer fremden Entität und bindet den Nutzer an ein System, aus dem es keinen praktikablen Ausweg gibt.

## Axiome
- [[Der NAME (in Großbuchstaben) ist nicht identisch mit der natürlichen Person, sondern ein externes Konstrukt]]
- [[Der NAME gehört einer anderen Entität (Staat, System, Körperschaft)]]
- [[Die Nutzung des NAMENs begründet Unterwerfung unter das System]]
- [[Vollständige Teilhabe an der Gesellschaft ist ohne NAME unmöglich]]

## Begriffe

### NAME
**Englisch:** STRAWMAN / ALL-CAPS NAME
**Deutsch:** Großbuchstaben-Name / Strohmann-Konstrukt
Der in Großbuchstaben geschriebene Identifikator, der bei der Geburt registriert wird und als juristische Fiktion von der natürlichen Person getrennt existiert

### Falle
**Englisch:** Trap / Joinder
**Deutsch:** Falle / Zwangsbindung
Zustand erzwungener Systemteilnahme ohne echte Ausstiegsoption

### Nutzen
**Englisch:** Use / Joinder
**Deutsch:** Nutzung / Beitritt
Aktive Verwendung des NAMENs in Rechtsgeschäften und gesellschaftlichen Transaktionen, die als Zustimmung zur Unterwerfung gilt


## Argumentationsketten
- NAME gehört einer fremden Entität → Nutzung des NAMENs = Anerkennung fremder Herrschaft (weil Eigentumslogik: wer etwas nutzt, unterliegt dem Eigentümer)
- Nutzung = Unterwerfung → Nicht-Nutzung = Befreiung (weil Umkehrschluss der Eigentumslogik)
- Nicht-Nutzung → vollständiger gesellschaftlicher Ausschluss (weil alle Systeme auf NAME-Identifikation aufbauen)
- Vollständiger Ausschluss bei Nicht-Nutzung → das System ist eine Falle ohne echten Ausweg (weil beide Optionen negative Konsequenzen haben)

## Offene Stellen
- Wer genau ist 'denen'? Die Entität, der der NAME gehört, bleibt undefiniert (Staat, Bank, Vatikan, Körperschaft?)
- Wie wird Eigentumsanspruch am NAME rechtlich oder faktisch begründet? Der Mechanismus fehlt.
- Der Text benennt das Dilemma, bietet aber keine Lösung: Wenn Nutzung Falle und Nicht-Nutzung Ausschluss bedeutet, bleibt die Argumentation ohne Ausweg
- Interne Spannung: Wenn es keine praktikable Alternative gibt, verliert die Warnung vor der 'Falle' ihren handlungsanleitenden Wert
- Warum konstituiert Nutzung automatisch Zustimmung? Die Freiwilligkeits-Prämisse wird nicht begründet
