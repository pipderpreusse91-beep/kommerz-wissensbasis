# Analyseprinzip

Alle Inhalte werden ausschließlich nach interner Logik bewertet — kein Abgleich mit externen Rechtssystemen oder Wahrheitsansprüchen.

## Was geprüft wird
- Welche Grundaxiome werden vorausgesetzt?
- Wie werden Schlüsselbegriffe intern definiert?
- Folgen Schlussfolgerungen logisch aus den Axiomen?
- Wo entstehen interne Widersprüche?
- Wo bleibt das System eine Antwort schuldig?
