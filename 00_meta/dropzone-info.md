# Dropzone — Anleitung

Lege hier Dateien ab die in die Wissensbasis eingearbeitet werden sollen.

## Akzeptierte Formate
| Format | Inhalt |
|--------|--------|
| `.md` / `.txt` | Texte, Mitschriften |
| `.pdf` | Dokumente |
| `.mp3` / `.ogg` / `.m4a` | Audio (wird automatisch transkribiert) |

## Verarbeitung starten
```
python ~/kommerz_pipeline/scripts/05_dropzone_processor.py
```
Verarbeitete Dateien werden nach `Archiv/` verschoben.
